# 测试说明文档

## 1. 测试简介

针对后端服务器的代码进行黑盒和白盒测试

## 2. 测试目的

- 通过对测试结果的分析，得到对软件质量的评价
- 评估测试执行和测试计划是否符合
- 分析系统存在的缺陷，为修复和预防bug提供建议

## 3. 测试工具

- Postman来进行服务的功能性测试
- Jmeter来进行接口压力测试
